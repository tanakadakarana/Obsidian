オブジェクトの生成に関係するのは、以下の５パターンです。 Factory Methodパターン インスタンスの生成をサブクラスに任せます。その結果、インスタンスの生成を柔軟に行うことができます。 Abstract Factoryパターン 関連したり依存しあうオブジェクト群のインスタンスを生成することができます。 Builderパターン 複雑なインスタンスを段階的に作成します。 Prototypeパターン Prototypeとは試作品という意味ではなく、ここでは原型という意味です。原型のオブジェクトをいくつか用意しておき、それらをコピーしてインスタンスを生成します。 Singletonパターン インスタンスが１つだけしか存在しないクラスを作ります。 — location: [196](kindle://book?action=open&asin=B086D2HLKK&location=196) ^ref-19438
